# Visual Workflow Pipeline Builder

A full-stack web application to create and manage node-based workflows using a visual interface. Users can build pipelines by connecting nodes and validate them as Directed Acyclic Graphs (DAG).

## 🚀 Tech Stack
- Frontend: React.js (JavaScript)
- Backend: Python (FastAPI)

## ▶️ How to Run

### 1. Frontend
```bash
cd frontend
npm install
npm start
